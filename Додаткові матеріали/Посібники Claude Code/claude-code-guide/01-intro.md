# 1. Вступ

Цей посібник є продовженням мої статті [Мій досвід з Claude Code після 2 тижнів пригод](https://sankalp.bearblog.dev/my-claude-code-experience-after-2-weeks-of-usage/) з липня 2025 року.

## Короткий огляд попереднього матеріалу

Якщо ви новачок у Claude Code або просто хочете освіжити свої знання, рекомендуємо прочитати той посібник. Там ми охопили:

- **CLAUDE.md** - як структурувати ваші персональні інструкції
- **Скетчпад** - інструмент для тимчасових нотаток та планування
- **Інструмент завдання** (тепер суб-агенти) - як делегувати складні завдання
- **План + виконання** - основний робочий процес в Claude Code
- **Управління контекстним вікном** - як не переповнити контекст
- **Компактація** - як скоротити багатослівні розмови
- **Скорочення** (`/resume`) - як перезапустити розмову
- **Пам'ять** - основи системи запам'ятовування
- **Спеціальні комбінації** (！та Shift + ?) - швидкі доступи

## Що нового в цьому посібнику

Цей посібник фокусується на:

1. **Claude Code 2.0** - більш детальне дослідження нових функцій
2. **Opus 4.5** - чому цей модель змінив гру
3. **Контекстна інженерія** - як оптимізувати токени і контекст
4. **Суб-агенти** - глибоке розуміння того, як вони працюють
5. **Команди та гаки** - як автоматизувати свій робочий процес
6. **Навички та плагіни** - новій спосіб організації знань
7. **Комбіновані стратегії** - як мішати всі ці інструменти разом

## Цільова аудиторія

Цей посібник створений для:

- **Розробників**, які активно використовують Claude Code
- **Нетехнічних людей** (technially-lite), які тільки починають з агентів
- **Дослідників**, які цікавляться інженерією контексту та промптування
- **Всіх, хто хоче розуміти, як працюють кодуючі агенти**

## Ключовий посил

> **Контекстна інженерія та промптування більше не дива - це навмисна практика, яку можна вивчити й вдосконалити.**

Як каже Карпатій, технологія розвивається неймовірно швидко. Але замість того, щоб просто "тримати крок", кращий підхід - **збагатити себе** за допомогою цих інструментів:

1. **Залишайтесь оновленими** з новими версіями інструментів
2. **Розвивайте свій домен** - навчайтеся глибше в вашій галузі
3. **Гарайте більше** - експериментуйте та розвивайте інтуїцію

Цей посібник допоможе вам з усіма трьома напрямками.
